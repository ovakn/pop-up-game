Pop Up - веб-игра, целью которой является набор наибольшего количества очков путём лопанья шаров. Клиентская часть написана на TypeScript с использованием Pixi.js, сервер - на Java с netty.

# Геймплей
Шары вылетают с нижней части экрана, поднимаются и в определённый момент начинают падать; каждый упавший шар забирают одну жизнь у игрока (при начале игры даётся 3 жизни).
### Виды шаров
Обычный (зелёный) - при тапе добавляет 1 очко\
Опасный (красный) - забирает одну жизнь\
x2 (золотой) - удваивает получаемые очки на 10 секунд\
Замедление (голубой) - замедляет время на 5 секунд\
Лечение (белый) - добавляет жизнь, количество не может быть больше 3\
Щит (фиолетовый) - даёт игроку возможность ошибиться и не потерять жизнь\
Бомба (чёрный) - взрывает все обычные шары и добавляет очки за них\

# Инструкция по запуску
## Запуск сервера:
Скачать и открыть папку сервера. Дальше:\
mvn clean package\
mvn exec:java -Dexec.mainClass="com.example.Main"\
*На компьютер должны быть установлены Java и Maven*\

## Запуск клиента
Также открыть папку клиента, затем:\
npm install\
npm run dev\

<b>Игра запущена и может быть открыта в браузере по адресу http://localhost:3000/</b>
